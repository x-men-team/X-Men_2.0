PowerShell commands for packaging the application:

1. Jlink command to create a runtime image:

jlink --module-path "$env:JAVA_HOME\jmods;.\javafx-libs" --add-modules java.base,java.compiler,java.datatransfer,java.desktop,java.instrument,java.logging,java.management,java.naming,java.net.http,java.rmi,java.security.jgss,java.sql,java.xml,javafx.base,javafx.graphics,javafx.controls,javafx.media --output myruntime


2. Command to test the application using the runtime image:

 .\myruntime\bin\java -jar target\X-Men_2.0-0.0.1-SNAPSHOT.jar


3. Command to build the executable file for the application:

jpackage --type exe --name "X-Men" --input target --main-jar X-Men_2.0-0.0.1-SNAPSHOT.jar --runtime-image .\myruntime --app-version 1.0.0 --win-shortcut --win-menu --win-dir-chooser --win-console

4. Command to remove the runtime image:

Remove-Item -Recurse -Force .\myruntime
